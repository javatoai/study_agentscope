家宴菜单 — 炖鸡版四人餐 + 龙虾

采购清单

类别食材
肉类走地鸡 半只
海鲜龙虾
蔬菜青椒 2个、生菜 2颗、土豆或香菇
蛋类鸡蛋 4-5个（2个炒、2个做汤）
干货紫菜（干圆饼）、八角
香辛料姜、蒜、葱
调料确认生抽、老抽、蚝油、盐、十三香、油、香油

做菜顺序

  1. 炖鸡先上锅（全程 40-50 分钟）
  2. 炖鸡空隙洗菜切菜
  3. 青椒炒鸡蛋
  4. 蒜蓉生菜
  5. 炒龙虾
  6. 炖鸡收汁出锅
  7. 紫菜蛋花汤 — 最后做

一、炖鸡

  1. 鸡块冷水下锅，煮开撇浮沫，捞出冲洗
  2. 热油 → 姜蒜爆香 → 鸡块炒到表面微黄
  3. 生抽 + 老抽（多放点上色）+ 十三香一小勺，炒匀
  4. 倒热水没过鸡，扔八角。大火烧开转小火，盖盖炖 40 分钟
  5. 加盐 → 大火收汁 → 关火放蚝油

二、青椒炒鸡蛋

  1. 鸡蛋打散炒熟盛出
  2. 热油 → 蒜爆香 → 青椒炒软
  3. 鸡蛋倒回 → 生抽 → 盐 → 关火放蚝油

三、蒜蓉生菜

  1. 水烧开，生菜烫 20 秒捞出装盘
  2. 热油 → 蒜末爆香 → 关火 → 加生抽 + 蚝油 + 一点水搅匀
  3. 浇在生菜上

四、炒龙虾

  1. 龙虾刷洗干净，剁成大块（钳子拍裂方便入味）
  2. 热油（油比平时炒菜多一点）→ 姜蒜爆香
  3. 龙虾块下锅大火翻炒到壳变红
  4. 生抽 + 一点点水，盖盖焖 3-5 分钟
  5. 开盖 → 盐 → 大火收干汤汁 → 关火放蚝油 → 撒葱花

五、紫菜蛋花汤

  1. 水烧开，紫菜撕碎丢进去
  2. 鸡蛋打散，汤沸腾时慢慢倒入
  3. 关火 → 盐 + 香油 + 葱花
